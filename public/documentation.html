This API will provide users with access to information about various movies, directors, and genres based on an undecided theme/genre. Uesrs will be able to sign up, update their personal information, and create a list of their favorite movies.

Post
Business Logic URL HTTP Req Body Res Body
Get a list of movies /movies Get None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] Get None A JSON object holding data about a single movie, containing the name, director, and genre
Example:

{ "_id": objectId,
"Title": "Movie Title",
"Description": "Movie Description",
"Genre":
"Name": "Genre's Name",
"Description": "Genre's Description",
"Director":
"Name": "Director's Name",
"Bio": "Director's Bio",
"Actors": ["Actor A", "Actor B"],
"imageUrl": "www.movielink.com",
"Featured": Boolean }

Get a list of directors /directors Get None A JSON object holding data about all the directors
Get data about a single director by name /directors/[Name] Get None A JSON object holding data about a single director, containing a bio
Example:

{
"Name": "Director's Name", "Bio": "Director's Bio" }

Get a list of genres /genres Get None A JSON object holding data about all the genres
Get data about a single genre /genres/[Name] Get None A JSON object holding data about a single genre, containing a description
Example:

{
"Name": "Genre's Name", "Description": "Genre's Description" }

Add a user /users A JSON object holding data about the user to add, structured like:
Example:

{ "Name": "Princess Peach", "Username": "princesspeach", "Password": "peachy1519", "Email": "peachy1519@gmail.com", "Birthday": "02/02/1997" }

A JSON object holding data about the user that was added, including an ID:
Example:

{ "_id": ObjectId, "Name": "Princess Peach", "Username": "princesspeach", "Password": "peachy1519", "Email": "peachy1519@gmail.com", "Birthday": "02/02/1997" }

Update a user /users/[Username] Put A JSON object holding data about the user to update
Example:

{
"Email": "peachpuff1519@gmail.com" }

A JSON object holding successfully updated user data
Example:

{ "_id": ObjectId, "Name": "Princess Peach", "Username": "princesspeach", "Password": "peachy1519", "Email": "peachpuff1519@gmail.com", "Birthday": "02/02/1997" }

Add movies to user's favorites /users/[Userame]/movies[MovieID] Post None A JSON object holding data about the user's favorite movie
Example:

{ "_id": ObjectId, "Name": "Princess Peach", "Username": "princesspeach", "Password": "peachy1519", "Email": "peachpuff1519@gmail.com", "Birthday": "02/02/1997" "Favorite Movies": [MovieID] }

Remove movies from user's favorites /users/[Userame]/movies[MovieID] Delete None A text message indicating whether the movie was successfully removed
Delete user account /users/[Name] Delete None A text message indicating whether the user was successfully removed
